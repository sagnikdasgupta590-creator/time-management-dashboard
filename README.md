# Project Name: time-management-dashboard
# overview: Web-based time management dashboard for visualising the user activity providing feedback using JSON.
# Features:
- Reads user activity data from a JSON file  
- Displays time usage in a visual format  
- Provides feedback based on time spent
# Technologies Used:
- HTML  
- CSS (Bootstrap)  
- JavaScript
# Purpose: To help students better understand how they spend their time and improved time management habits.
# How it works: The application loads time data from a JSON file, processes it using JavaScript, and displays the results on a web page. Based on the data, the system provides simple feedback to the user.
